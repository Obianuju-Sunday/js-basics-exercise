# JS Basics Exercises

This repository contains exercises and mini-projects covering the fundamentals of JavaScript:
- Arrays
- Loops
- Functions
- Methods
- Objects

## Purpose
These exercises were completed as part of my JavaScript learning journey, helping me practice core concepts and improve coding fluency.

## How to Use

- Run them in the browser console or Node.js to see results.
- Modify the exercises to test and experiment with new ideas.
